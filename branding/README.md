# Chime brand assets

- `chime-logo.svg` / `chime-logo.png` — full wordmark
- `chime-mark.svg` / `chime-mark.png` — standalone "C" mark (favicon, app icon, avatar use)

SVGs are the source of truth; PNGs are 2000x2000 exports for places that need a raster.
Also mirrored under `web/public/brand/` so the dashboard can reference them directly
(e.g. `/brand/chime-logo.svg`).
